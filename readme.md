# 🔐 Simulation de Piratage 🖥️

# Bienvenue sur le projet **Simulation de Piratage** ! 🎉 Ce projet recrée une simulation de piratage fictive dans un terminal. Entrez un mot de passe et suivez les étapes du processus de piratage ! C'est une simulation amusante pour démontrer des concepts de piratage de manière interactive. 🚀

---

# 📋 Fonctionnalités :
# - 🖥️ **Interface de terminal** : Un terminal stylisé où chaque étape de piratage est affichée comme dans un vrai terminal.
# - ⏳ **Effet de frappe** : Les messages apparaissent comme si quelqu'un tapait chaque caractère à la fois, rendant l'expérience plus immersive.
# - ⚠️ **Simulation d'erreurs et de tentatives** : Des erreurs aléatoires et des tentatives de correction sont simulées pour rendre l'expérience plus réaliste.
# - 🔓 **Découverte du mot de passe** : Une fois que le mot de passe est entré, la simulation affiche des étapes pour "trouver" et "décoder" le mot de passe.

---

# 
# 🧑‍💻 Développement :
# Ce projet utilise uniquement des fichiers **HTML**, **CSS**, et **JavaScript**. Pas besoin de serveur ou de backend pour faire fonctionner ce projet, tout est géré côté client.

---

# 🎨 Personnalisation :
# - Si vous souhaitez personnaliser le style du terminal, vous pouvez modifier le fichier `style.css` pour changer la couleur du texte, la police, ou les animations.
# - Le fichier `script.js` gère la simulation. Vous pouvez ajouter plus d'étapes ou personnaliser le texte affiché pendant la simulation.

---

# 📝 Exemple de Fonctionnement :
# Voici un aperçu de ce que vous verrez lors de l'exécution du projet :

# ```bash
# Tentative de piratage en cours...
# Connexion au serveur distant...
# Erreur ! Impossible d'injecter le code dans le réseau.
